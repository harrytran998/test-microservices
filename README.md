# Test microservices

Uses the following technologies:

- Docker (and Docker Compose)
- React
- Node.js
- PostgresQL
- Sequelize

## Run Local

```sh
# in main directory
docker-compose up

# in a separate terminal, inside client-app
yarn && yarn watch
```
